# Monitoring Setup - Quick Reference

## ✅ Google Analytics Setup

### Step 1: Get Your Google Analytics ID

1. Go to [Google Analytics](https://analytics.google.com/)
2. Create a new property or use existing one
3. Get your Measurement ID (format: `G-XXXXXXXXXX`)

### Step 2: Add to Your Dashboard

1. Go to **Dashboard** → **Website** → **SEO Settings**
2. Find **"Google Analytics ID"** field
3. Enter your ID (e.g., `G-XXXXXXXXXX`)
4. Click **Save**

### Step 3: Verify

1. Visit your website
2. Open browser DevTools → Network tab
3. Look for requests to `google-analytics.com` or `googletagmanager.com`
4. Check Google Analytics dashboard - you should see real-time visitors

**That's it!** Google Analytics is now tracking your site automatically.

## 💡 Tips

1. **Google Analytics**: Data may take 24-48 hours to fully populate
2. **Real-time Data**: Check the "Realtime" section in Google Analytics to see immediate traffic
3. **Reports**: Use Google Analytics reports to track:
   - Page views
   - User behavior
   - Traffic sources
   - Conversion rates
   - Audience demographics
