# Public Assets

Place your static assets here, including:

## Logo
- **logo.png** - Your main logo file (recommended size: 240x80px or larger, transparent background)
- The logo will be used in:
  - Header (main navigation)
  - Footer
  - Admin Dashboard Sidebar

### Logo Specifications:
- Format: PNG (with transparency) or SVG
- Recommended dimensions: 240x80px (will scale to 32px height)
- File name: `logo.png` or `logo.svg`
- Place in: `/public/logo.png` or `/public/logo.svg`

## Hero Background Image
- **hero-bg.jpg** - Hero section background image for the homepage
- The image will be used in:
  - Homepage hero section (first section)

### Hero Background Specifications:
- Format: JPG or PNG
- Recommended dimensions: 1920x1080px or larger (full-width landscape)
- File name: `hero-bg.jpg` or `hero-bg.png`
- Place in: `/public/hero-bg.jpg` or `/public/hero-bg.png`
- The image will be displayed with a dark overlay (50% opacity) for better text readability

If you don't have these images yet, you can:
1. Add your logo file as `logo.png`
2. Add your hero background image as `hero-bg.jpg`
3. Or update the components to use different paths/names
